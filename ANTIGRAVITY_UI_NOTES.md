# Antigravity UI Notes

- **Goal**: Create high-end design system primitives for MAMILAS Pro OS without interfering with Codex's migration logic.
- **Dependencies**: None currently requested. We rely on React 19, framer-motion, lucide-react, and pure CSS.
- **Design Philosophy**: Deep black background, single gold accent (`--gold`), 1px hairline borders, glass/blur surfaces, purposeful motion (framer-motion).
- **Paths Owned**: `src/ui/**`, `src/styles/tokens.css`, `src/lab/**`, `lab.html`.

## Status
- Initializing tokens and motion presets.
- Primitives to be built: Button, Card, Panel, Chip, Stat, Field, Select, Tabs, Toast, Tooltip, ProgressRail, SwatchStrip, SceneCard, PreviewStage.
