## HOW TO READ A MAMILAS BRIEF

1. `== RENDER LOCK ==` → paste verbatim at the top of every image prompt.
2. `== REFERENCE DNA → DIRECTIVES ==` → take CAMERA / LIGHT / STAGING / TEXTURE.
3. `== PALETTE AS LIGHT ==` → read as light behavior, never flat fills.
4. `== SCENE DOSSIER ==` each [N] scene:
   - CONCEPT → dominant element
   - EVENT → motion seed event sentence
   - CAMERA → camera/vantage line
5. DNA avoid list → Negative line.
6. One prompt per scene. Total prompts = dossier scene count.
